Erstellt eine Tabelle mithilfe einer Vorlage

Eingaben

payload Array von Objekten
Die Daten, die verwendet werden, um die Zeilen zu erstellen. Jedes Objekt im Array stellt eine Tabellenzeile dar, und jeder Schlüssel des Objekts ist eine Spalte der Tabelle.
Daten string
Das Feld, das verwendet wird, um die Daten zum Erstellen der Tabelle zu bestimmen. Wenn msg.data nicht existiert, werden die Zeilenwerte aus msg.payload verwendet. Der Wert, der mit msg.data festgelegt wird, hat Vorrang gegenüber dem Wert im Formular der Node-Eigenschaften und msg.payload .
templateFilepath String
Der lokale Dateipfad zu einer .xlsx | .xls -Vorlage. Der Wert, der mit msg.templateFilepath festgelegt wird, hat Vorrang vor dem im Eigenschaftsformular des Knotens eingestellten Wert.
templateSheetName String
Der Name des Tabellenblatts, das aus der Vorlage verwendet werden soll. Der Wert, der mit msg.templateSheetName festgelegt wird, hat Vorrang vor dem im Eigenschaftsformular des Knotens eingestellten Wert.
outputFilepath String
Der lokale Dateipfad, um die generierte .xlsx | .xls -Datei zu speichern. Der Wert, der mit msg.outputFilepath festgelegt wird, hat Vorrang vor dem im Eigenschaftsformular des Knotens eingestellten Wert.

Ausgaben

  1. Ausgabepfad
    payload String
    Der Pfad, unter dem die generierte Datei gespeichert wurde.

Details

Das folgende Beispiel zeigt den Payload, der an den Knoten gesendet werden soll. Jedes Objekt im Array entspricht einer Zeile, und wenn eine Spalte leer ist, sollten Sie einen leeren String übergeben. 


msg.payload = [ 
    { 
        "column-1" : "wert", 
        "column-2" : "wert", 
        "column-3" : "wert" 
    },
    { 
        "column-1" : "wert", 
        "column-2" : "", 
        "column-3" : "wert" 
    }
]

Wenn Sie eine der Knoten-Eigenschaften überschreiben müssen, legen Sie diese einfach im msg -Objekt fest, wie im folgenden Beispiel gezeigt. 


msg = {
    "payload": [
        { 
            "column-1" : "wert", 
            "column-2" : "wert", 
            "column-3" : "wert" 
        },
        { 
            "column-1" : "wert", 
            "column-2" : "", 
            "column-3" : "wert" 
        }
    ],
    "templateFilepath": "meine-vorlage.xlsx",
    "templateSheetName": "mein blattname",
    "outputFilepath": "ausgabe.xlsx"
}
