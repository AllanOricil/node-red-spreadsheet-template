Creates a spreadsheet using a template

Inputs

payload array of objects
the data used to create the rows. Each object in the array represents a spreadsheet row. And each object key is a column of the spreadsheet.
templateFilepath string
the local filepath to a .xlsx | .xls template. The value set using msg.templateFilepath has precedence over the value set in the node properties form.
templateSheetName string
the name of the spreadsheet sheet to use from the template. The value set using msg.templateSheetName has precedence over the value set in the node properties form.
outputFilepath string
the local filepath to store the generated .xlsx | .xls file. The value set using msg.outputFilepath has precedence over the value set in the node properties form.

Outputs

  1. Output Filepath
    payload string
    the path where the generated file was saved to.

Details

The following example demonstrates the payload to be sent to the node. Each object within the array corresponds to a row, and if a column is empty, you should pass an empty string. 


msg.payload = [ 
    { 
        "column-1" : "value", 
        "column-2" : "value", 
        "column-3" : "value" 
    },
    { 
        "column-1" : "value", 
        "column-2" : "", 
        "column-3" : "value" 
    }
]

If you need to overwrite any of the node properties, just set them in the msg object, as shown below. 


msg = {
    "payload": {
        { 
            "column-1" : "value", 
            "column-2" : "value", 
            "column-3" : "value" 
        },
        { 
            "column-1" : "value", 
            "column-2" : "", 
            "column-3" : "value" 
        }
    },
    "templateFilepath": "my-template.xlsx",
    "templateSheetName": "my sheet name",
    "outputFilepath": "output.xlsx"
}
