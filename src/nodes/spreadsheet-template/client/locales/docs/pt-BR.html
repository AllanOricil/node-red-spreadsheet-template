Cria uma planilha usando um modelo

Entradas

payload array de objetos
os dados usados para criar as linhas. Cada objeto no array representa uma linha da planilha. E cada chave do objeto corresponde a uma coluna da planilha.
dados string
O campo utilizado para definir os dados usados para renderizar a planilha. Quando msg.data não existe, os valores das linhas vêm de msg.payload . O valor definido usando msg.data tem precedência sobre o valor definido no formulário de propriedades do nó e msg.payload .
templateFilepath string
o caminho local para um arquivo de modelo .xlsx | .xls . O valor definido usando msg.templateFilepath tem precedência sobre o valor definido no formulário de propriedades do nó.
templateSheetName string
o nome da planilha a ser usada a partir do modelo. O valor definido usando msg.templateSheetName tem precedência sobre o valor definido no formulário de propriedades do nó.
outputFilepath string
o caminho local para armazenar o arquivo gerado .xlsx | .xls . O valor definido usando msg.outputFilepath tem precedência sobre o valor definido no formulário de propriedades do nó.

Saídas

  1. Caminho do Arquivo de Saída
    payload string
    o caminho onde o arquivo gerado foi salvo.

Detalhes

O exemplo a seguir demonstra o payload a ser enviado para o nó. Cada objeto dentro do array corresponde a uma linha, e se uma coluna estiver vazia, você deve passar uma string vazia. 


msg.payload = [ 
    { 
        "column-1" : "valor", 
        "column-2" : "valor", 
        "column-3" : "valor" 
    },
    { 
        "column-1" : "valor", 
        "column-2" : "", 
        "column-3" : "valor" 
    }
]

Se você precisar sobrescrever qualquer uma das propriedades do nó, basta defini-las no msg conforme mostrado abaixo. 


msg = {
    "payload": [
        { 
            "column-1" : "valor", 
            "column-2" : "valor", 
            "column-3" : "valor" 
        },
        { 
            "column-1" : "valor", 
            "column-2" : "", 
            "column-3" : "valor" 
        }
    ],
    "templateFilepath": "meu-modelo.xlsx",
    "templateSheetName": "nome da minha planilha",
    "outputFilepath": "saida.xlsx"
}
